Project name:

Timestamp API

Example Usage:

Get Current Data

To get the data for the current date, go to the route:

/api/

Get Other Dates

To get the data for a specific date, add a parameter of a natural language date or a unix timestamp to the set route. Natural language dates must be written in YYYY-MM-DD formatting such as:

/api/2000-09-13

Unix timestamps must be a valid set of numbers such as:

/api/968803200000

Shows Error

If you specify invalid string such as :

/api/timestamp/invalidstring

Example Output:

{ "unix": 968803200000, "utc": "Wed, 13 Sep 2000 00:00:00 GMT" }
